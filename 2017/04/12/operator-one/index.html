你真的理解三目运算符吗？

发表于 2017-04-12

运算符可谓是语言的基础，看书时感觉so easy，可有时碰到一些运算符的复杂运算就不知所云，之所谓懂其内涵才能游刃有余。

三目运算符：
对于条件表达式b?x:y，先计算条件b，然后进行判断。如果b的值为true，计算x的值，运算结果为x的值；否则，计算y的值，运算结果为y的值。一个条件表达式从不会既计算x，又计算y。条件运算符是右结合的。

下面看一些例子

String s = "aaa";
String a = "f:" + s != null ? s : "";
System.out.println(a);

a?b:c?d:e

int a = 2;
int result = (a++ > 2)?(++a):(a+=3);

int a = 3;
int result = a += 2 > 1 ? ++a : a;
System.out.printf("\nresult = %d \n", result);

分析：

  1. 第一题，运算符有+,!=,?由于优先级为+优先!=优先?,所以会先计算”f:” + s，然后再判断是否为真，最后进行三目运算。 最后结果为”aaa”

  2. 只有?运算符，这里考察结合性，由于单目、三目、赋值为右结合，所以这会先计算c?d:e,得到结果后在进行a?b:x计算

  3. 从左到右，运算符有(), ++, >, ?，这里优先级为++优于>优于？所以先进性a++运算,可以理解为2 > 2为false，这时a = 3，然后计算a+=3,所以最后结果为6
  4. 这里从左到右计算符为：+=,>, ? 优先级是>优于？优于+=,所以2 > 1永远为true，这里编辑器会进行优化处理，优化后的结果为result = a++a, 结果为7
